# crustaceans.sh

A postcard-themed guide to the wider OpenClaw ecosystem.

`crustaceans.sh` is a small Eleventy site focused on orientation: what the major Claws are, how they differ, where to start, and which links are worth following first.

## What it covers
- ecosystem overview for OpenClaw, NanoClaw, PicoClaw, and NemoClaw
- newcomer-friendly routing and comparison notes
- curated links to docs, repos, community resources, and use cases
- field-guide style recommendations distilled from public discussion and documentation

## Project goals
- help a cold visitor understand the landscape quickly
- keep the information practical, curated, and memorable
- use a postcard / beach-desk visual language without sacrificing legibility

## Local development
```bash
npm install
npm run dev
```

Useful commands:
- `npm run check` — build and verify the homepage output exists
- `npm run deploy` — build and deploy with Wrangler

## Stack
- Eleventy
- Cloudflare Workers via Wrangler
